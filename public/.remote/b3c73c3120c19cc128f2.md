# 0.初めに
どういうわけかMacBookProを1円で入手したので延命したいということでやっていきます。
今回はインストールも作成も同じMacで行いますので少し手順が変わります
# 1.諸注意
この記事で取り扱う方法はAppleの公式が推奨しないものであるため、
行う場合は自己責任でお願い致します。
また、一部モデルではUSB2.0のハブとキーボード、マウスが必要となります
# 2.準備するもの
- USBメモリ(32GB以上)
- MacOS10.10以降が動作するMac
- インターネット接続
- 時間
- 根気
#  3.インストールの準備
まず手始めにディスクユーティリティでUSBをフォーマットしておきます
```
フォーマット: Mac OS拡張（ジャーナリング）
方式: GUIDパーティションマップ
```

次に、

https://github.com/dortania/OpenCore-Legacy-Patcher/releases/latest

ここからOpenCore-Patcher.pkgというファイルをDLします
その後、インストール。
OpenCore-Legacy-Patcher(以下OLCP)
を起動すると、
Build and Install OpenCoreという項目があるので、
実行、その後内臓ディスクにインストール
その後、OpenCoreをインストールするEFIパーティションを選択します。
Reboot to Applyと出ればOpenCoreのインストールは完了です。

# 4.インストーラーの作成
いよいよインストーラーを作ります。
まずはOCLPを起動してCreate macOS Installerを選択。
現在ダウンロード可能な最新バージョンのインストーラーが表示されるので、お好みで選択し、「Download」をクリックします。
ダウンロード、保存が済むとUSBの選択が出ます。
その後、選択したUSBの内容は消えますという趣旨のメッセージが出るのでyesを押します
そしてさらに待つと
successfully.....と出るのでyesを押します
済んだら、先ほどのインストールのようにOLCPをUSBにもインストールします。
完了すると再起動するか聞かれますがignoreを押します
# 5.インストール
まずはMacの電源を落とします
次にoptionキーを押しながら電源ボタンを押して、ブートピッカーを表示、
先ほどのOCLPのアイコンのEFIbootを選択
その次はインストーラーのディスクを選択し起動
その後は流れに沿ってインストールし、セットアップを済ませます。
最後に、OCLPを更新し、post install root patchを実行して完了です。
