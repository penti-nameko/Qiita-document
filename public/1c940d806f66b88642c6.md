# 0.アーキテクチャとは
プロセッサの基本的な設計構造や命令セットの体系を指す。
x86 x64 は CISC(Complex Instruction Set Computer) と呼ばれる命令セット。
「複雑な処理」を「少ない命令」で実行しようとする思想である。
arm は RISC(Reduced Instruction Set Computer) と呼ばれる命令セットとなる。
# 1.ARM,x86の違い
勘違いしやすいが、ARMとx86(x64)は「会社」の違いではなく「命令セット(アーキテクチャ)」の違いである。
ちなみにIntelとAMDはどちらもx86(x64)アーキテクチャを採用しているメーカーで、CPU市場では競合関係にあるが、命令セットという土台自体は共通している。
ARMはもともと省電力性を重視した設計で、その特性からスマートフォン向けとして広く普及してきたが、近年はApple SiliconやAWS Graviton、Snapdragon X等、PCやサーバー用途にも採用範囲が広がっている。
# 2.x64,x86とは
x64は64bitの情報を処理可能。
x86は32bitの情報が処理可能、
となっている。
因みに、
`amd64`や`x86_64`
等と書いてあってもx64とほぼ同じだと思って問題はない。
基本的にARMとx86(x64)プロセッサは互換性がないので、アプリのダウンロード時に選択する場合は注意すべきである。
# 3.終わりに
意外とわかりにくかったりするようでそうでもないので覚えておきたい基礎知識でした。
それでは良いインフラライフを！
